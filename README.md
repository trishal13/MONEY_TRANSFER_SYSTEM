# Money Transfer System

A responsive web-based application that allows users to simulate secure money transfers, view transaction records, and manage their accounts. The application also features an admin dashboard for comprehensive management.

## Features

- **User Login/Registration**: User verification and account creation.
- **Money Transactions**: Simulates fund transfers between users with PIN verification for added security.
- **Transaction Records**: A comprehensive log of all user transactions.
- **Check Balance**: Users can verify their account balance using a PIN.
- **Admin Dashboard**: Powered by Django Admin, enabling CRUD operations for users and transactions.

## Technologies Used

- **HTML & CSS**: For creating a responsive and user-friendly interface.
- **Python**: Backend logic and server-side scripting.
- **Django**: Web framework for rapid development and management.
- **Jinja**: Template engine for dynamic content rendering.
- **MySQL**: Database for storing user and transaction data.

### Prerequisites

- **Python 3.x**: Ensure Python is installed. Download from [python.org](https://www.python.org/).
- **MySQL**: Ensure MySQL server is set up. Download from [mysql.com](https://www.mysql.com/).
- **Django**: Install Django using pip.
